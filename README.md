# Алгоритмы и структуры данных. Set 3. Задача A2. Экспериментальные данные

Данный репозиторий содержит результаты экспериментального исследования производительности двух алгоритмов сортировки:

- **Merge Sort** — классический алгоритм сортировки слиянием
- **Hybrid Sort** — гибридный алгоритм (Merge Sort + Insertion Sort), где для малых подмассивов используется сортировка вставками

---

## Типы исследуемых входных данных

Для оценки эффективности алгоритмов использовались три вида входных массивов:

| Тип массива | Описание |
|------------|-----------|
| **Random** | Массив случайных чисел |
| **Inverted** | Полностью отсортирован в обратном порядке |
| **Almost Sorted** | Почти отсортирован, содержит небольшие отклонения |

Каждый эксперимент проводился с массивами размера **от 500 до 100000**, с шагом **1000**.  
Каждое измерение повторялось **7 раз**, затем усреднялось.

## Содержимое репозитория

| Файл | Описание |
|------|----------|
| **sorting_results.csv** | Сводные данные по времени выполнения для всех типов входных массивов |

---

## Формат данных в CSV

Каждая строка соответствует одному размеру массива `N` и содержит времена выполнения в наносекундах:

| Поле | Тип | Описание |
|------|-----|----------|
| **N** | int | Размер массива |
| **Merge_Random** | int | Время Merge Sort на случайных данных |
| **Hybrid_Random** | int | Время Hybrid Sort на случайных данных |
| **Merge_Inverted** | int | Время Merge Sort на инвертированных данных |
| **Hybrid_Inverted** | int | Время Hybrid Sort на инвертированных данных |
| **Merge_Almost** | int | Время Merge Sort на почти отсортированных данных |
| **Hybrid_Almost** | int | Время Hybrid Sort на почти отсортированных данных |

---

## Назначение данных

Данные предназначены для:

- сравнения эффективности Merge Sort и Hybrid Sort
- анализа влияния структуры входных данных на производительность алгоритмов
- построения графиков зависимости времени выполнения от `N`:
  - для случайных массивов
  - для инвертированных массивов
  - для почти отсортированных массивов
